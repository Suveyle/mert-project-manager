# mert-project-manager
# 🚀 Project Manager App

Modern ve kullanıcı dostu bir **proje yönetim sistemi**.  
Görevleri, bütçeyi ve proje süreçlerini kolayca takip etmeni sağlar.

---

## 📌 Proje Hakkında

Bu uygulama, basit ama etkili bir proje yönetim arayüzü sunar.  
Amaç; görevleri düzenlemek, ilerlemeyi takip etmek ve proje sürecini görselleştirmektir.

---

## ✨ Özellikler

- 🗂️ Proje görevlerini listeleme
- ➕ Yeni görev ekleme
- ✏️ Görev düzenleme ve silme
- 📊 İlerleme takibi
- 💾 LocalStorage ile veri saklama
- 📱 Responsive (mobil uyumlu) tasarım

---

## 🛠️ Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage
- Chart.js (grafik sistemi)
- Tailwind CSS (CDN)
